# 🎮 小游戏合集

一个纯前端实现的经典小游戏合集，使用 HTML、CSS 和 JavaScript 构建，无需任何框架或构建工具，打开即玩。

## 游戏列表

本项目包含 **10 款**经典小游戏：

| 游戏 | 描述 | 操作方式 |
|------|------|----------|
| **2048** | 经典数字合并益智游戏 | 方向键 / 触屏滑动 |
| **贪吃蛇** | 经典贪吃蛇 | 方向键 |
| **俄罗斯方块** | 经典方块消除游戏 | ← → 移动，↑ 旋转，↓ 加速 |
| **打砖块** | 经典弹球打砖块 | ← → 方向键 |
| **井字棋** | 双人对战井字棋 | 鼠标点击 |
| **记忆翻牌** | 考验记忆力 | 鼠标点击 |
| **打地鼠** | 考验反应力 | 鼠标点击 |
| **推箱子** | 经典益智推箱子 | 方向键 |
| **数字猜谜** | 猜数字小游戏 | 键盘输入 |
| **消消乐** | 三消宝石游戏 | 鼠标点击交换 |

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式与动画（Grid 布局、渐变背景、过渡动画）
- **JavaScript（ES6+）** - 游戏逻辑（Canvas 绘制、类封装、事件处理）
- **LocalStorage** - 最高分持久化存储

## 快速开始

### 在线游玩

直接将项目文件部署到任意静态服务器即可，或使用以下方式本地运行：

```bash
# 使用 Python 启动本地服务器
python -m http.server 8000 --bind 127.0.0.1

# 或使用 Node.js
npx serve .
```

然后在浏览器中打开：
- **游戏合集入口**：[http://localhost:8000/games.html](http://localhost:8000/games.html)
- **2048 独立页面**：[http://localhost:8000/index.html](http://localhost:8000/index.html)

### 直接打开

由于项目为纯静态文件，也可以直接在浏览器中打开 `games.html` 或 `index.html` 文件。

## 项目结构

```
2048game/
├── index.html          # 2048 独立游戏页面
├── games.html          # 游戏合集入口页面（包含全部 10 款游戏）
├── style.css           # 2048 独立页面样式
├── site.css            # 游戏合集页面样式
├── game.js             # 2048 游戏核心逻辑（独立版本）
├── site.js             # 游戏合集框架 + 全部 10 款游戏逻辑
├── LICENSE             # Apache 2.0 开源协议
├── README.md           # 项目说明
└── .gitignore          # Git 忽略配置
```

## 特色功能

- **响应式设计**：适配桌面端和移动端，支持触屏操作
- **平滑动画**：方块移动、合并、弹出等均有流畅过渡动画
- **分数持久化**：最高分自动保存至浏览器本地存储
- **模块化架构**：每款游戏独立封装为类，便于维护和扩展
- **模态框切换**：游戏合集页面采用弹窗模式，无需页面跳转
